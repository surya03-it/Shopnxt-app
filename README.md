# Shopnow-app
A simple ecommerce web app built with Python/Flask featuring a Sign In page and Product Index page. Containerized with Docker and deployed on AWS ECS Fargate via a fully automated CI/CD pipeline using CodePipeline, CodeBuild, and CodeDeploy (Blue/Green).
